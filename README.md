# Start-Up-Dashboard-
📊 Startup Funding Analysis in India (2015–2021)
🔍 Overview

This project analyzes sector-wise startup funding trends in India between 2015 and 2021 using an interactive dashboard and supporting economic analysis.

The objective is to identify how venture capital allocation has shifted across sectors, with a particular focus on the contrasting trajectories of:

Fintech (rapid growth)
E-commerce (stagnation)

🎯 Objectives
Examine trends in startup funding across sectors over time
Identify high-growth sectors and stagnating sectors
Explain funding patterns using economic reasoning and policy context
Understand investor behavior in emerging markets like India

📁 Dataset
Sector-wise startup funding data (2015–2021)
Variables include:
Sector (Fintech, E-commerce, etc.)
Funding amount (USD)


Data Sources:
Kaggle

🛠️ Tools & Technologies
R Programming
Basic data cleaning and aggregation techniques
Data visualization and dashboarding

📊 Dashboard Features
Sector-wise funding comparison (2015 vs 2021)
Trend visualization across years
Identification of top-funded sectors
Comparative analysis between fintech and e-commerce

📈 Key Findings

1. Fintech Funding Surge
Fintech funding increased significantly, reaching $4.41 billion by 2021
Growth driven by:
Financial inclusion expansion
Digital payment infrastructure (UPI)
Increased investor confidence in scalable financial models

2. E-commerce Funding Stagnation
E-commerce funding remained relatively stable across the period
Despite early dominance, growth plateaued due to:
Market maturity
High competition
Low profit margins
